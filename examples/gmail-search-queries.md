Useful Gmail Cleanup Queries
Large emails:
```text
larger:10M
```
Old attachments:
```text
has:attachment older_than:2y
```
Old promotions:
```text
category:promotions older_than:1y
```
Old social emails:
```text
category:social older_than:1y
```
Old updates:
```text
category:updates older_than:2y
```
Sender/domain cleanup:
```text
from:(example.com) older_than:1y
```
